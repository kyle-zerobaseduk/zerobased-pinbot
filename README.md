# 🤖 PinBot - Automated Pinterest Scheduler

Automated Pinterest pin posting with AI-powered content generation for your Etsy shop.

## Features

✅ **24/7 Automated Posting** - Runs continuously on Railway or any Node.js server  
✅ **AI-Generated Descriptions** - Uses Claude Haiku to write unique pin copy  
✅ **Scheduled Slots** - Post at specific times (e.g., 9am, 1pm, 6pm)  
✅ **Smart Rotation** - Cycles through products and images to avoid spam detection  
✅ **Cockpit Dashboard** - Dark theme, real-time monitoring and control  
✅ **Zero Coding** - No Git or command line needed for setup  

## Quick Setup (5 mins)

### 1. Gather Your Credentials

You should have these saved in Notepad:
- **PINTEREST_TOKEN** - From Pinterest Developer App
- **ANTHROPIC_API_KEY** - From Anthropic Console
- **Dashboard Password** - Something secure (e.g., `MyPinBot2024!`)
